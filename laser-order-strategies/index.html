Scan Strategy
Deterministic Hilbert Curve

A space-filling curve that visits every point in a continuous "wiggle".

Superior locality preservation. Avoids long linear fronts and preserves structural integrity.
Deterministic Tri-Phase Sweep

Processes main diagonal, then LL sweep, then UR corner inward.

Rationale: Hardware default (e.g. xTool). Path is mathematically fixed and repeatable.
Deterministic Diagonal Wave

Baseline wave approach. Creates a moving thermal front.

Calculated based on coordinate distance. Fixed result.
Stochastic Quadrant Leapfrog

Jumps between zones with randomized cell fills.

Reduces repeatable warping by shuffling processing order within quadrants.
Deterministic Horizontal Raster

Baseline linear pathing. High thermal overlap.

Pathing is strictly sequential and fixed.
Stochastic Dispersive (Bit-Reverse)

Uses Bayer scattering to jump between non-adjacent cells.

The macro-strategy is fixed, but micro-pathing contains shuffling to avoid resonance.
