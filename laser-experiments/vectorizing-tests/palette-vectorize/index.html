Palette Vectorizer

Palettize first, then vectorize each color as a separate SVG layer — clean flat fills that xTool Studio can select for laser layer separation.

Why palettize before vectorizing? Standard vectorizers trace edges in a single greyscale pass — colors blend into each other and you get one messy combined path. This tool runs k-means color quantization first, snapping every pixel to its nearest palette color. Then it traces each palette color independently, producing clean non-overlapping regions. The SVG output uses separate <g fill="#rrggbb"> groups — one per color — which xTool Studio reads as distinct layers you can assign to separate engraving passes with different power and speed.
Palettize → Per-Color Vectorize → Multi-Layer SVG

Using in xTool Studio. Import the downloaded SVG. Each fill color appears as a separate layer in the Layers panel. Select a layer → assign laser parameters (power, speed, passes). Use darker colors for deeper engrave passes, lighter for surface marking. Black is typically the deepest engraving layer.
Tips for best results. More colors = more layers but smaller per-layer regions. For logos and illustrations, 4–6 colors usually gives the clearest separation. Increase the Simplify slider for cleaner paths with fewer points (better for cutting); decrease it to preserve fine detail (better for engraving). Pre-blur 1–2 smooths noisy JPEG artifacts before palettizing.
