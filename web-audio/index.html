Dynamic Audio on the Web

The Web Audio API lets you build audio processing graphs entirely in JavaScript — oscillators, filters, effects, and precise timing, all without plugins.

Everything is modelled as a directed graph of AudioNodes connected together. A source node (oscillator, buffer, microphone) feeds through processing nodes (filters, gain, effects) to the AudioContext.destination — your speakers. Nodes are created from a single AudioContext and time is controlled via its high-precision currentTime clock.
